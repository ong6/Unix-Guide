About

This is a Summary of all the basic Unix command that I use often designed for CS2030/S students. This guide will cover most of the basic Unix tools that you will need to speed up and make writing code on Unix systems a breeze.

Refer to the notes before looking at the cheatsheet to get a basic understanding of Unix.
